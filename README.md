## Run Locally

Clone the project

```bash
  git clone https://github.com/MatriX1232/so_long.git
```

Go to the project directory

```bash
  cd so_long
```

Run Makefile

```bash
  make
```

Run project
```bash
  ./so_long <PATH/TO/MAP.ber>
```
